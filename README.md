# MediaPipe Fall

一个基于 **YOLOv8**、**MediaPipe** 与 **PyQt6** 的桌面端人员跌倒检测系统。它支持摄像头和本地视频输入，实时识别画面中的人员姿态；在检测到疑似跌倒时自动触发告警，并将事件片段保存下来供回放。

> 本项目适用于原型验证、课程设计和智能安防场景探索；检测结果不应作为医疗诊断或唯一的安全决策依据。
# 项目概述：MediaPipe Fall

MediaPipe Fall Detector 是一套面向室内安防、养老看护和视频分析原型场景的实时跌倒检测桌面应用。项目将 YOLOv8 的人员检测能力与 MediaPipe 的人体姿态估计结合，通过分析肩部和髋部的相对位置计算躯干倾角，从而识别人员是否处于站立、跌倒中或平躺状态。

系统支持摄像头实时画面和本地视频文件两种输入方式。检测到连续的疑似跌倒状态后，程序会在界面中显示告警、累计告警次数，并自动将事件过程录制为视频片段保存，方便后续复核。PyQt6 图形界面提供视频预览、状态面板、运行日志以及历史告警片段回放功能。

## 核心价值

- **实时性**：逐帧进行人员检测与姿态分析，适合实时监控原型。
- **可追溯性**：自动保留告警视频片段，便于人工复核。
- **易用性**：桌面图形界面减少命令行操作，支持摄像头和本地视频导入。
- **可扩展性**：可替换 YOLO 模型、调整姿态阈值，或扩展消息通知、事件数据库和多摄像头管理等功能。

一、 判定跌倒的核心依据
系统判断跌倒的核心依据是人体躯干（Torso）的倾斜角度。
- 原理：当人正常站立时，肩膀中心与臀部中心连线接近垂直于地面；当人跌倒或平躺时，躯干会发生大幅度倾斜，与地面的夹角变小或与垂直方向的夹角增大。
- 阈值判定：
  - 站立 (Normal)：躯干倾斜角小于设定阈值（如 $20^\circ$）。
  - 跌倒中 (Falling)：躯干倾斜角处于过渡区间。
  - 平躺 (Lying)：躯干倾斜角大于设定阈值（如 $50^\circ$）。
二、 核心算法
1. 目标检测算法 (YOLOv8)：由 Ultralytics 开发的 YOLOv8 模型，用于在每一帧视频中精准框出人体（类别 ID 为 0），排除背景干扰，锁定检测区域（ROI）。
2. 人体骨骼关键点提取算法 (MediaPipe Pose)：Google 开源的 MediaPipe 姿态估计框架，能够实时回归出人体躯干的关键节点（特别是左右肩膀、左右髋关节等关键坐标）。
3. 几何角度计算算法 (math.atan2)：利用数学中的反正切函数，根据肩膀中心点和髋关节中心点的二维坐标差值，计算出躯干向量与垂直方向的夹角。

## 适用边界

本项目采用基于躯干角度的轻量级规则判断，容易受到遮挡、镜头角度、人员姿势、光照和多人重叠等因素影响。它适合作为计算机视觉项目原型或辅助告警工具，正式部署前应在目标场景中进行数据验证、阈值校准和人工复核流程设计。

## 功能特性

- YOLOv8 实时定位视频画面中的人员
- MediaPipe Pose 提取人体肩部、髋部等关键点
- 根据躯干倾角识别站立、跌倒中与平躺姿态
- 支持默认摄像头与 `.mp4`、`.avi`、`.mov` 本地视频
- 跌倒告警时自动保存视频片段至 `result/`
- 提供告警次数、实时状态、运行日志及历史片段回放界面

## 技术原理

1. 使用 YOLOv8 检测每帧中的人员（COCO 类别 `person`）。
2. 对人员区域进行 MediaPipe 姿态估计。
3. 以左右肩膀与左右髋部的中点计算躯干倾角。
4. 根据阈值判断姿态：小于 20° 为站立，大于 50° 为平躺，介于两者之间视为跌倒中。
5. 当跌倒状态连续出现约 0.1–0.5 秒时触发告警，并写入事件视频；姿态恢复后结束录制。

## 环境要求

- Windows 10/11（已在 Windows 环境开发）
- Python 3.8–3.11
- 摄像头（使用实时检测时需要）

## 快速开始

```bash
git clone https://github.com/<your-username>/Mediapipe-Fall-Detector.git
cd Mediapipe-Fall-Detector

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

将 YOLOv8 Nano 权重文件放在项目根目录，并命名为 `yolov8n.pt`。然后运行：

```bash
python main.py
```

首次使用也可以从 [Ultralytics YOLOv8](https://github.com/ultralytics/ultralytics) 获取 `yolov8n.pt`。若网络可用，Ultralytics 通常会在权重缺失时尝试自动下载。

## 使用说明

1. 启动程序后，选择“切换为摄像头”或“导入视频文件”。
2. 点击“启动实时检测”。
3. 在右侧查看当前姿态、告警状态、告警次数和运行日志。
4. 检测到跌倒后，告警片段会保存到 `result/`。
5. 点击“历史跌倒片段”可在程序内查看已保存的视频。

## 注意事项

- 本项目默认使用摄像头索引 `0`；如有多个摄像头，可在 `main.py` 中调整。
- 程序在 Windows 下会优先使用 `msyh.ttc` 或 `simhei.ttf` 绘制中文；若中文显示异常，请安装相应字体或修改字体路径。
- 姿态阈值、连续帧判定时长会影响灵敏度和误报率，建议针对实际安装高度、拍摄角度和场景进行调参。
- 建议不要将 `result/`、大体积测试视频或模型权重提交到 Git 仓库；可通过 Releases、网盘或模型仓库分发它们。
